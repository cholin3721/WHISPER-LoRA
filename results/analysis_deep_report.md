## 5. 발표용 헤드라인 (이번 주 슬라이드 후보)

1. **실측 CER 6.85% — 문헌 추정(20~30%)의 1/3 수준**
   → Whisper-large-v3는 한국어 표준어 노인 발화를 이미 잘 인식한다. PPT의 출발 가정 자체가 갱신 대상.

2. **H2 가설(Deletion 지배)은 데이터로 강하게 지지됨**
   → Del 4.3% vs Sub 1.7% vs Ins 0.1%. Del:Sub = 2.5배, Del:Ins = 43배.
   → "긴 침묵·느린 발화로 인한 단어 누락"이라는 음향학적 가설이 실측으로 확인됨.

3. **화자간 격차 3배 (3.25% ~ 9.52%)**
   → "노인 음성"을 단일 그룹으로 다루는 가정의 한계 노출.
   → VOTE400의 구음장애·초고령 화자군에서는 격차가 더 클 것으로 예상.

4. **H1 재정의**
   → 기존: "Whisper가 노인 음성에서 무너진다 → LoRA로 메운다"
   → 수정: "Whisper는 표준 노인 발화는 잘하지만 어려운 subgroup에서 무너진다 → LoRA가 그 격차를 선택적으로 메운다"

5. **다음 단계**
   → 1순위: VOTE400 협약 진행 + 구음장애/80+ subgroup baseline 재측정
   → 2순위: AI허브 샘플 위에서 LoRA 파이프라인 코드 검증 (학교 GPU 열리는 대로 본 학습)
